
# Cyber Security Project 4 – Audited Systems: Blue Team Defense

##  Project Description
This project focuses on performing a basic security audit of a Windows-based system from a Blue Team perspective. The objective is to identify common security weaknesses, evaluate the overall security posture, and provide recommendations to improve system security.

The audit includes:
- System Information Review
- Password Policy Assessment
- Windows Update Status
- Firewall Configuration
- Microsoft Defender Assessment
- Installed Software Review
- Risk Assessment
- Security Recommendations

##  Technologies / Tools Used
- Microsoft Windows 11
- Windows Security
- Windows Defender Firewall
- Microsoft Defender Antivirus
- System Settings
- Microsoft Word (Documentation)

##  Project Contents
- Security Audit Report
- Risk Assessment
- Security Findings
- Recommendations
- Screenshots of Security Settings

##  How to Run
This is a documentation-based cybersecurity project and does not require any programming or execution.

To review the project:
1. Open the project folder.
2. Read the Security Audit Report.
3. Review the findings and recommendations.
4. View the attached screenshots as evidence of the audit.

##  Learning Outcomes
- Understand the Blue Team approach to system security.
- Perform a basic security audit.
- Identify vulnerabilities in a Windows system.
- Assess risks based on security findings.
- Recommend best practices to improve system security.
